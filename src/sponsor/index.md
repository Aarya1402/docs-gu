# Vue.js સ્પોન્સર બનો (Become a Vue.js Sponsor) {#become-a-vue-js-sponsor}

Vue.js એ MIT લાઇસન્સ પ્રાપ્ત ઓપન સોર્સ પ્રોજેક્ટ છે અને વાપરવા માટે સંપૂર્ણપણે મફત છે.
આટલી મોટી ઇકોસિસ્ટમ જાળવી રાખવા અને પ્રોજેક્ટ માટે નવી સુવિધાઓ વિકસાવવા માટે જરૂરી પ્રચંડ પ્રયત્નો અમારા સ્પોન્સર્સના ઉદાર નાણાકીય સહયોગને કારણે જ વ્યવસ્થિત રીતે ચાલુ રહી શકે છે.

## સ્પોન્સર કેવી રીતે બનવું (How to Sponsor) {#how-to-sponsor}

સ્પોન્સરશિપ [GitHub Sponsors](https://github.com/sponsors/yyx990803) અથવા [OpenCollective](https://opencollective.com/vuejs) દ્વારા કરી શકાય છે. ઇન્વોઇસ GitHub ની પેમેન્ટ સિસ્ટમ દ્વારા મેળવી શકાય છે. માસિક-આવર્તક (monthly-recurring) સ્પોન્સરશિપ અને એક-વખતનું દાન બંને સ્વીકારવામાં આવે છે. આવર્તક સ્પોન્સરશિપ [Sponsorship Tiers](#tier-benefits) માં ઉલ્લેખિત લોગો પ્લેસમેન્ટના હકદાર છે.

જો તમારી પાસે ટિયર્સ (tiers), પેમેન્ટ લોજિસ્ટિક્સ અથવા સ્પોન્સર એક્સપોઝર ડેટા અંગે પ્રશ્નો હોય, તો કૃપા કરીને [sponsor@vuejs.org](mailto:sponsor@vuejs.org?subject=Vue.js%20sponsorship%20inquiry) પર સંપર્ક કરો.

## વ્યવસાય તરીકે Vue ને સ્પોન્સર કરવું (Sponsoring Vue as a Business) {#sponsoring-vue-as-a-business}

Vue ને સ્પોન્સર કરવાથી તમને અમારી વેબસાઇટ અને GitHub પ્રોજેક્ટ READMEs દ્વારા વિશ્વભરના **૨ મિલિયનથી** વધુ Vue ડેવલપર્સ સુધી પહોંચવાની તક મળે છે. આનાથી માત્ર સીધા ગ્રાહકો જ નથી મળતા, પણ ઓપન સોર્સની કાળજી રાખતા વ્યવસાય તરીકે તમારી બ્રાન્ડની ઓળખ પણ સુધરે છે. ડેવલપર્સ માટે પ્રોડક્ટ્સ બનાવતી કંપનીઓ માટે આ એક અમૂર્ત પરંતુ અત્યંત મહત્વપૂર્ણ સંપત્તિ છે, કારણ કે તે તમારા કન્વર્ઝન રેટમાં સુધારો કરે છે.

જો તમે આવક પેદા કરતી પ્રોડક્ટ બનાવવા માટે Vue નો ઉપયોગ કરી રહ્યાં હોવ, તો Vue ના વિકાસને સ્પોન્સર કરવું વ્યવસાયિક દ્રષ્ટિએ સમજદારીભર્યું છે: **તે ખાતરી કરે છે કે પ્રોજેક્ટ કે જેના પર તમારી પ્રોડક્ટ આધાર રાખે છે તે સ્વસ્થ રહે અને સક્રિયપણે જાળવવામાં આવે.** Vue સમુદાયમાં એક્સપોઝર અને સકારાત્મક બ્રાન્ડ ઈમેજ પણ Vue ડેવલપર્સને આકર્ષવા અને રિક્રૂટ કરવાનું સરળ બનાવે છે.

જો તમે એવી પ્રોડક્ટ બનાવી રહ્યા છો કે જેમાં તમારા લક્ષિત ગ્રાહકો ડેવલપર્સ છે, તો તમને સ્પોન્સરશિપ એક્સપોઝર દ્વારા ઉચ્ચ ગુણવત્તાવાળો ટ્રાફિક મળશે, કારણ કે અમારા તમામ મુલાકાતીઓ ડેવલપર્સ છે. સ્પોન્સરશિપ બ્રાન્ડ ઓળખ પણ બનાવે છે અને કન્વર્ઝનમાં સુધારો કરે છે.

## વ્યક્તિગત તરીકે Vue ને સ્પોન્સર કરવું (Sponsoring Vue as an Individual) {#sponsoring-vue-as-an-individual}

જો તમે વ્યક્તિગત વપરાશકર્તા છો અને Vue ના ઉપયોગથી તમારી ઉત્પાદકતાનો આનંદ માણી રહ્યા છો, તો પ્રશંસાના ચિહ્ન તરીકે દાન કરવાનું વિચારો - જેમ કે અમને ક્યારેક કોફી પીવડાવવી. અમારા ટીમના ઘણા સભ્યો GitHub Sponsors દ્વારા સ્પોન્સરશિપ અને દાન સ્વીકારે છે. અમારા [ટીમ પેજ](/about/team) પર દરેક ટીમના સભ્યની પ્રોફાઇલ પર "Sponsor" બટન શોધો.

તમે તમારા એમ્પ્લોયરને વ્યવસાય તરીકે Vue ને સ્પોન્સર કરવા માટે સમજાવવાનો પણ પ્રયાસ કરી શકો છો. આ કદાચ સરળ ન હોય, પરંતુ વ્યવસાયી સ્પોન્સરશિપ સામાન્ય રીતે વ્યક્તિગત દાન કરતા OSS પ્રોજેક્ટ્સની ટકાઉપણું પર ઘણી મોટી અસર કરે છે, તેથી જો તમે સફળ થશો તો તમે અમને ઘણી વધુ મદદ કરશો.
